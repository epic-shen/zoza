## «zoza» module ##

### Информация ###

**«zoza»** — простой модуль для создания ботов Telegram.

Основан на telegraf-develop версии.

***

### Установка ###

Для установки модуля в консоли напишите:

```js

npm i zoza

```

### Использование ###

После установки пакета:

```js

const { zoza } = require('zoza')

const bot = new zoza('ТОКЕН_БОТА')

```

Запускаем бота

```js

bot.run()

```
